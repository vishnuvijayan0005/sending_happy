# Ayurveda Health Landing Page

A clean, responsive landing page for *Ayurveda Health* service offering — built with modern web technologies and deployed on Vercel.

# Live Demo

You can view the live site here:  
[https://sending-happy-theta.vercel.app/](https://sending-happy-theta.vercel.app/)

## Features

- Hero section with a bold heading and call-to-action.  
- Services section listing key capabilities (Web Development, UI/UX Design, App Development, DevOps).  
- Simple navigation (Home, Top, Bottoms, Jump Suits/Rompers — placeholder items).  
- Fully responsive design (desktop → mobile).  
- Social links in the footer.  
- Easy customisation for branding & content.

## Built With

- HTML5 & CSS3   
- Responsive layout 
- Deployed with Vercel (via `vercel.app`)
